INSTRUCTIONS

To change the bind address in MySQL, you can modify the "bind-address" setting in the MySQL configuration file .

Step 1 :-  Open the MySQL configuration file in Terminal or command prompt

Step 2 :-  Look for the "bind-address" setting and modify its value to the desired IP address or hostname.
For example, if you want to bind MySQL to all available network interfaces, use "0.0.0.0" or we can use "%" symbol.

Step 3 :-  Save the changes and exit the Terminal.

Step 4 :-  Create Wildcard user in mysql.
